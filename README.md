# Trading Strategy Analytics API

A backend API for tracking and analyzing trading performance.

## 🚀 Features
- JWT Authentication
- Trade Logging (CRUD)
- Profit/Loss Calculation
- Strategy Analytics
- Monthly Performance Tracking
- Swagger API Docs

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Swagger
